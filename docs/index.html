Vocal-Burst Annotation Pipeline

Detect & caption vocal bursts (laughs, sighs, gasps, moans, screams, coughs, sneezes, …) in audio — an ensemble of laion/vocalburst-locator (detector) and laion/vocalburst-captioning-whisper (captioner).

🎧 Interactive demo & benchmark report — audio samples + predictions + Gemini-3.1-Pro scores for the best detection thresholds (0.88 / 0.89 / 0.85).
🔬 Captioner comparison — old sound-effect captioner vs the fine-tuned vocal-burst captioner on the same detections.
